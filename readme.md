## Генератор Паролей

Этот простой генератор паролей позволяет создавать надежные пароли заданной длины. Программа запрашивает у пользователя желаемое количество символов для пароля и предлагает включить специальные символы. 

**Особенности:**
*   **Настраиваемая длина:** Пользователь может указать желаемую длину пароля (минимум 5 символов).
*   **Включение специальных символов:** Возможность добавить специальные символы (!@#$%^&*) для повышения сложности.
*   **Оценка сложности:** После генерации пароля программа оценивает его сложность (Слабый, Средний, Сильный, Очень сильный) на основе длины и наличия различных типов символов (буквы в верхнем и нижнем регистре, цифры, специальные символы).
*   **Использование `secrets` и `random`:** Для обеспечения криптографической стойкости и случайности символов используются модули `secrets` и `random`.

### Как использовать:
1.  Запустите программу.
2.  Введите желаемое количество символов для вашего пароля (минимум 5).
3.  Ответьте 'да' или 'нет' на вопрос о включении специальных символов.
4.  Получите сгенерированный пароль и его оценку сложности.
