# pass_gen
Программа разработана для генерации безопасных паролей с оценкой их сложности.
ИИ использован для улучшения алгоритма оценки.
В программе реализовано:
Генерация паролей с настраиваемыми параметрами.
Создана оценка сложности паролей.
Добавлена возможность сохранения паролей в зашифрованном виде.
Использован ИИ-инструмент для улучшения алгоритма оценки сложности.
Реализован простой интерфейс.
Написаны тесты для проверки генерации и оценки паролей.
Создана документация по использованию.
